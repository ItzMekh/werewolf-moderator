## 🚀 Roadmap & Ideas (รายละเอียดแผนการพัฒนาและไอเดียเพิ่มเติม)

จากระบบที่มีอยู่ แนะนำให้พิจารณาเพิ่มฟีเจอร์เหล่านี้เพื่อทำให้เกมน่าสนใจและตอบโจทย์ผู้เล่นมากขึ้น รวมถึงการเตรียมพร้อมสำหรับการสเกลในอนาคต:

### 🌟 1. การเพิ่มบทบาทใหม่ๆ (New Roles Expansion)
เกม Werewolf จะสนุกขึ้นเมื่อมีบทวิเศษที่หลากหลาย เพื่อให้มีความซับซ้อนและบลัฟกันมันส์ขึ้น
- **🫙 Witch (แม่มด):** มียาชุบชีวิต 1 ขวด และยาพิษ 1 ขวด ตลอดเกม
- **🔫 Hunter (นายพราน):** เมื่อถูกฆ่าตอนกลางคืนหรือโหวตตอนกลางวัน สามารถเลือกยิงปืนลากใครตายไปด้วยได้ 1 คนทันที
- **💘 Cupid (กามเทพ):** ในคืนแรก สามารถเลือกคู่รัก 2 คน หากคนหนึ่งตาย อีกคนต้องตายตาม
- **🃏 Fool / Jester (คนบ้า):** เงื่อนไขชนะคือ "ต้องทำให้ตัวเองถูกโหวตตายในตอนกลางวัน"

### 🛡️ 2. ยกระดับระบบเกมให้มีความอึด (Resilience & Connection)
- **💾 Persistent Game State (Redis/DB):** ปัจจุบันเกมอยู่ใน Memory (`rooms` ใน `index.js`) หากเซิฟเวอร์รีสตาร์ทห้องจะหายหมด การนำ Redis มาคั่นจะทำให้สานต่อเกมเก่าได้ 100% แม้เซิฟเวอร์ดับ
- **📱 QR Code เข้าร่วมห้องเร่งด่วน:** ในหน้ารอเริ่มของ Moderator ควรมี QR Code แสดงขึ้นมา เพื่อให้ผู้เล่นใช้มือถือสแกนและทะลุเข้าเกมได้เลย

### 🎮 3. ยกระดับ Player Experience (UX/UI & Immersive)
- **📳 Haptic Feedback (ระบบสั่นบนมือถือ):** ใส่สั่นด้วย `navigator.vibrate([200, 100, 200])` เวลาถึงเทิร์นตัวเองในตอนกลางคืน ผู้เล่นจะได้ไม่ต้องจ้องจอตลอดเวลา
- **🎵 Sound Effects & BGM (ระบบเสียง):** แอดเสียงบรรยากาศหมาป่าหอนลงไปบนคอมเครื่องหลัก

### 🤖 4. โหมด Automator (AI/System Moderator)
หากผู้เล่นมีน้อย เปลี่ยนให้ Server ทำหน้าที่แพลนเวลาให้และประกาศเสียงพูดอัตโนมัติ (Text-to-Speech) แทนที่คนที่ต้องเป็นจับเกม

### 📊 5. สถิติเชิงปริมาณและระบบความสำเร็จ (Leaderboard & Titles)
เพิ่มแรงจูงใจในการสร้าง Account ด้วยการทำ Leaderboard และให้ฉายาหลังจบเกม
